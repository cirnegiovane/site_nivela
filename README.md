# Nivela UERJ

![Vercel Deployment](https://img.shields.io/badge/Deployed%20on-Vercel-black?style=for-the-badge&logo=vercel)
![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)

O **Nivela UERJ** é um projeto de extensão da Universidade do Estado do Rio de Janeiro (UERJ) que nasceu com o objetivo de tornar o aprendizado de Cálculo mais acessível. O foco principal é o suporte a alunos calouros e veteranos na disciplina de Cálculo I, fornecendo a base necessária através do conceito de **"Cálculo Zero"**.

---

## Identidade Visual e Conceito

A identidade visual do projeto foi pensada para refletir o universo das exatas de forma leve, lúdica e dinâmica. Fugindo da rigidez tradicional da matemática, o site utiliza formas orgânicas e uma paleta de cores vibrante para criar uma experiência acolhedora.

### Tipografia

- **Logo e Títulos:** [Forum](https://fonts.google.com/specimen/Forum) (Remete à lógica e elegância clássica).
- **Corpo de Texto:** Century Gothic (Garante acessibilidade e clareza na leitura).

### Paleta de Cores

- **Verdes (Lógica e Crescimento):** `#27722E`, `#23922D`, `#3DB649`, `#8AD192`, `#CAFFD0`
- **Laranjas (Energia e Diálogo):** `#ED7508`, `#FF8D26`, `#FFA656`, `#FFBA7C`, `#FFDFC3`

---

## Tecnologias Utilizadas

- **[React 18](https://react.dev/):** Biblioteca para construção de interfaces.
- **[Vite](https://vitejs.dev/):** Ferramenta de build rápida e moderna.
- **[Tailwind CSS 4](https://tailwindcss.com/):** Estilização baseada em utilitários e variáveis CSS.
- **[React Router 7](https://reactrouter.com/):** Gerenciamento de rotas e navegação.
- **[Formspree](https://formspree.io/):** Integração para envio de e-mails via formulário.

---

## Funcionalidades

- **Cálculo Zero:** Material de revisão de base matemática (Álgebra, Funções, Trigonometria).
- **Repositório de Listas:** Acesso rápido a arquivos PDF de exercícios preparados pela equipe.
- **Interface Responsiva:** Otimizado para dispositivos móveis e desktops.
- **Formulário de Contato:** Canal direto entre alunos e a coordenação do projeto.

---

## 🛠️ Como rodar o projeto localmente

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/seu-usuario/site_nivela.git
   ```

2. **Entre na pasta do projeto:**

   ```bash
   cd site_nivela
   ```

3. **Instale as dependências:**

   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
